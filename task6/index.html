Задача 6. Светофор модифицированный.

Представлены 3 черных круга. При клике на каждый из кругов цвет будет меняться в следующем порядке:
Первый черый круг: цвет меняется на зеленый. Цвет остальных кругов не меняется.
Второй черный круг: цвет меняется на желтый. Первый круг меняет цвет на черный. Третий круг не меняет цвет, оставаясь черным.
Третий круг: цвет меняется на красный. Первый круг цвет не меняет. Второй круг меняет цвет на черный.
