Задача 2. Подсказка.

Примеры команд на JavaScript.

Нажмите на название команды для получения подсказки.

console.log

Пример записи команды:

console.log('Сообщение, которе будет выведено в консоли');

Alert()

Пример записи команды:

alert('Сообщение,которое будет показано во всплывающем окне ');

Prompt()

Пример записи команды:

prompt('Сообщение с полем ввода данных, которые надо заполнить и нажать OK');
